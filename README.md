## 🚀 Getting Started

### Prerequisites

Make sure you have **Node.js** and **npm** installed.

### Installation

1. Clone the repository:
   ```sh
   git clone git ...
   ```
2. Navigate to the project directory:
   ```sh
   cd your-portfolio
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
